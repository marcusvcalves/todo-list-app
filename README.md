# todo-list-app

Um CRUD de Lista de Tarefas com .NET no backend e React no frontend
## Tecnologias Utilizadas:
- .NET 6
- Entity Framework Core
- API Rest
- React
- Axios
- Styled Components

## Instalação
1. Tendo o docker instalado, basta executar o comando:
```
docker compose up --build
```
